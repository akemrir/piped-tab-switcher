# piped-tab-switcher

Addon shortcut ctrl+comma.

Only Linux is supported.

# Requirements:

* `dmenu` in $PATH
* `ruby 2.7`

To install, clone into an appropriate, non-temporary folder and run `install-native.sh` to
set up the manifest for the native backend.
